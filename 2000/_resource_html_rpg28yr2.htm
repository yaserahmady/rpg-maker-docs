A.Window background color (32 x 32)

B.Window frame color and arrows (32 x 32)

C.Frames for command cursor, 2 types (each 32 x 32)

D.Symbols for shop equipment displays, 4 types (each 8 x 8 pixels, x 4 frames of animation)

E.Background pattern for menu screens (16 x 16)

F.Text background color (16 x 16)

G.Numbers, symbols and space for timer display, 12 types (each 8 x 16)

H.Airship shadow frames, 2 types (each 16 x 16)

I.Text color, 20 types (each 16 x 16)

 

·Of the text colors specified in (i), the first 5 will be used as follows:

0: Default (No specification) text color

1: Color used for status names etc.

2: Increased parameter color

3: Decreased parameter color, disabled command color

4: Color used when HP, MP etc. are low

 

When changing text color during [Show Text] event commands, [Status] display color etc., the colors will be set with regards to this numbering.
