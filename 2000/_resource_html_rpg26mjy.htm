Switches

[Events] → [Switches]

 

·Switches are a type of data that remember two possible states: ON and OFF. They are used to mark whether a target has been met, such as game progress, actor's status, or some other condition.

·When the game starts, all switches are set to the OFF state.

·5000 switches can be used. Each switch has its own fixed ID number. Switches can be assigned any name.

·Switches can be specified by clicking the [...] button on the right side of the settings field to display the [Switch] window. The contents of each part of the window are as follows.

 

1.Block list

·Click to select the switches to display in the numbered list. The numbers in brackets are the range of switches, where the left side of the hyphen (-) denotes the start point and the right side the end point of the range.

 

2.Numbered list

·Displays switch numbers and names in groups of 20.

·When an entry on the list is clicked, the switch is selected and the name is highlighted. After selecting the switch, click the [OK] button or double click the entry to confirm.

·When an entry is right-clicked, a shortcut menu will be displayed. The commands in this menu can be used to [Copy], [Paste] or [Clear] the name of the switch.

·The [F4] and [F5] keys can be used to quickly change to the previous and next list options respectively, also across pages.

 

3.Name

·The name of the selected switch can be edited. The maximum length is 20 characters.

 

4.Maximum Number

·Changes the maximum number of switches available (up to 5000).

·To change this number, click the button to display the [Change Maximum] window, enter a new value, and click the [OK] button.
