Cell Batch Settings

[Database] → [Animations] → [Cell Batch Setting]

 

Changes all the same-numbered cells in the frame. The settings below can be changed in the [Cell Batch] window.

 

·Under [Frame #], check the boxes for the frames that should be set.

·Under [Cells], check the boxes for the cell numbers of the cells that should be affected. However, if the specified cells are not present in the given frames, no settings will be changed.

·Under [Pattern], specify the number of the cell frame for the given cell. Change the settings after the checkbox is enabled.

·Under [Magnification], specify the scale of the cell frame within a range of 20 to 800%. Change the settings after the checkbox is enabled.

·Under [Transparency], specify the opacity of the cell frame within a range of 0 to 100%. The greater the value, the more visible the background behind the cell will be. Change the settings after the checkbox is enabled.

·Under [Color], specify the tint color for the cell using the 4 components [Red (R)], [Green (G)], [Blue (B)] and [Saturation (S)]. With the original image as 100% in all components, specify the strengths of each component within a range of 0 to 200%. The adjusted colors will be displayed under [Sample]. Change the settings after the checkbox is enabled.
