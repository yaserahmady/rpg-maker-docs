Project

[Main Window] → [Menu Bar] → [Project] menu

New Project

·Creates a new project with one blank map and default data.

Open Project

·Opens an existing project.

 

Close Project

·Closes the current project in the editor.

·If the map data has not been saved, a confirmation window will appear to confirm if the data should be saved. If you want to save the data, click [Yes]; otherwise, click [No].
Clicking the [Cancel] button will return you to the editor without closing the project.

Create Game Installer

·Creates the data package for playing the game. There is an option to include the RTP, which is recommended in situations where the players will not have the program or RTP already installed.

 

Exit

·Closes the RPG Maker 2000 program.

·If the map data has not been saved, a confirmation window will appear to confirm if the data should be saved. If you want to save the data, click [Yes]; otherwise, click [No]. Clicking the [Cancel] button will return you to the editor without closing the project.
