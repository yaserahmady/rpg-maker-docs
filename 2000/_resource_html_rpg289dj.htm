Battle Event Setting

[Events] → [Battle Events] → [Battle Event Settings]

 

·Battle events are created and edited in the [Troops] tab in the [Database].

·Click to select the enemy group to set the battle event for from the list on the left side of the screen, and define the event content by specifying commands as required under the [Battle event] option.

·The options for settings are as follows.

 

·Create New / Copy / Paste / Delete

Operates on pages. Each button's function is similar to the New Page, Copy Page, Paste Page and Delete Page functions of map events.

 

1.Starting conditions

·Specify the conditions for starting the processing of content. This needs to be set in order for the battle event to start. If the condition is [None], a battle event will not be set for the enemy group.

·The event page will only be processed if all of the conditions are fulfilled.

·When conditions are fulfilled for multiple pages, unlike for map events, the content will be processed in order of lowest page number first.

·To set conditions, click the [...] button to open the [Conditions] window. Enable the relevant options from below, and set the targets and values.

 

 

Switch

Processing begins when a switch is ON.

Variable

Processing begins when a specified variable is above the given value.

Turn Number

When the specified turn count has been reached, processing will begin. The turn count is specified in the format Ax + B.
For example, if A = 3 and B = 1, valid turns are 1, 4, 7, and so on. After B turns, valid turns occur every A turns.
If set to 0 turns (A and B are both set to 0), content will be processed before any commands in battle are entered.

Party HP

If the actors' consumption level is within the specified percentage range, processing will occur. The consumption formula is as follows.
Consumption (%) = 100  100 x { (Total HP / Total maximum HP x 2 + Total MP / Total maximum MP) / 3 }

Enemy HP

When an enemy's HP percentage (%) is within the given range, processing will begin. Specify the enemy, the minimum, and maximum of the range.

Actor HP

When an actor's HP percentage (%) is within the given range, processing will begin. Specify the actor, the minimum, and maximum of the range.

 

2.(3) Event content

·Event content will be processed (1) as soon as the starting conditions are met, as well as (2) at the beginning of turns where the starting conditions are met.
