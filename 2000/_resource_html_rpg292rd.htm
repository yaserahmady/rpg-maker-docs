Common Settings

 

1.Name

·The name of the skill. The maximum length is 20 characters.

 

2.Skill Type

·Specify the skill category. From the dropdown menu, choose one of the following options:

oNormal: A normal skill which can be used for attacking, healing, inflicting a status effect, etc.

oTeleport: A skill which invokes the teleportation feature. When used (while teleportation is allowed), the teleportation menu will be displayed which allows the player to teleport to one of the saved places (which can be set through event commands).

oEscape: A skill which invokes the escaping feature. When used (while escaping is allowed), the player will be teleported back to the escape target position which can be set through event commands.

oSwitch: A skill which sets a switch to ON when used.

·Depending on the chosen category, the Category settings will change.

 

3.MP Cost

·This is the amount of MP that will be lost when the skill is used. Specify within a range of 0~999.

·In-game, if a character's current MP is less than the MP cost of a skill, the skill will not be usable.

·Alternatively, the MP cost can be set at a percentage value of the character's maximum MP. The MP spent on the skill will change depending on the maximum MP of the actor. This means that the higher the character's level, the more MP the skill will consume

 

4.Description

·The comment for the skill. In-game, this will be displayed when the skill is selected.

·Normal (If [Normal] is specified in 2)

 

5.Scope

Choose the skill target(s) from the dropdown menu.

 

Single Enemy

One of the opponents from the opposing group will be the target.

All Enemies

The entire opposing group will be the target.

The User

The User will be the target.

One Ally

One of the allies from the The User's group will be the target.

All Allies

The entire of the The User's group will be the target.

 

6. Battle Animation

·Choose the battle animation that will be displayed when the skill is used in battle, from the dropdown menu. The selected animation will be previewed.

 

7. Usage Message

·Enter the form of the message that will be displayed when the skill is used.

·Placeholders can be used.

oFor a description of placeholders in messages, please see [Terms].

oThe following placeholders can be used:

§%S: Name of the actor or enemy who casts the spell.

§%O: Target on which the spell is casted. This placeholder only works if the [Scope] is set to [Single Enemy], [The User] or [One Ally]!

§%U: Name of the spell. (This is an exception to the rule that the %U placeholder is used for units.)

·Clicking the casts, does, uses buttons automatically enters the texts "%S casts %U!", "%S does %U!" and "%S uses %U!" respectively.

 

8. Failed Message

·Enter the form of the message that will be displayed when the skill is used.

·The first line displays who used what skill. After the character's name, enter the text that represents the action. Clicking the cast, loosed, used buttons automatically enters the respective text into the sentence.

·The second line can contain any text that is a continuation of the first line.

9. Physical Rate

·Determines the influence of the The User's attack stat (includes the opponent's defense) on the skill's effect. Move the slider to specify the value within a range of 0~10.

·The smaller the value, the less effect the The User's attack has on the skill's effect.

11. Magical Rate

·Affects the variability of the effects of damage, HP recovered, skill effects etc. Move the slider to specify the value within a range of 0~10.

·For each level in precision, the range of values will increase by ±5%, up to a maximum of ±50%.

12. Variance

·Affects the variability of the effects of damage, HP recovered, skill effects etc. Move the slider to specify the value within a range of 0~10.

·For each level in precision, the range of values will increase by ±5%, up to a maximum of ±50%.

14. Stat Decrease (when effect range is [Single enemy] or [All enemies]) / Stat Increase (when effect range is [The User], [One Ally] or [All allies])

·Specify which stat the base effect value of the skill will be applied to. Enable the checkbox for each stat to be influenced.

·If [(Absorb)] is enabled, the stat reduction effect can only be applied to HP or MP. The reduced stat will be added to the The User's stat.

·If [(Ignore defence)] is enabled, the specified stat will be reduced without applying the opponent's defence.

·Reduced or increased stat values will only take effect during battle. Once the battle is over, the actor's stats will revert to normal.

14. Success Rate

·Specify the skill's success rate within a range of 0~100%. However, even if this value is set to 100%, the skill may still fail depending on the opponent's state.

15. State Infliction (when effect range is [Single enemy] or [All enemies]) / Heal States (when effect range is [User], [Single ally] or [All allies])

·Enable the respective checkbox to specify a [Status effect] that the skill can cause.

·When the target is Single enemy or All enemies, it will cause the specified [Status effect], and when the target is User or Single ally, it will heal the specified [Status effect].

16. Attack Element (when effect range is [Single enemy] or [All enemies]) / Elemental Defense (when effect range is [User], [Single ally] or [All allies])

·Enable the respective checkbox to specify a [Type] for the skill.

·If a skill with a [Type] that an opponent is weak to (has a high fluctuation rate) is used against it, the resulting effect value will be greater than normal.

17. Reduce Resistance(when effect range is [Single enemy] or [All enemies])/ Elemental Defense (when effect range is [User], [Single ally] or [All allies])

·If enabled, the resistance to the checked types in (16) of the target characters will decrease or increase by 1 stage.

·The resistance change is limited to the stages immediately above or below the character's original resistance. For example, if an actor has an elemental type resistance of C, the resistance level can change between B, C and D. Even if the resistance-changing skill is used multiple times, the type resistance will not exceed 1 stage above or below the original setting.

·Reduced or increased resistances will only take effect during battle. Once the battle is over, the actor's resistances will revert to normal.

 

·Transfer/Escape type (if [Transfer] or [Escape] is specified in 2)

18. Sound Effect

·Specify the sound effect to play when the skill is used. Click the [...] button to open the [Sound effects] window and choose the sound file to use from the list.

 

·Switch type (if [Switch] is specified in 2)

19. Occasion

·Specify whether the skill is usable in Field (on the map) and/or Battle scenarios respectively. The skill will be usable in the enabled scenarios.

20. Sound Effect

·Specify the sound effect to play when the skill is used. Click the [...] button to open the [Sound effects] window and choose the sound file to use from the list.

21. Usage Message

·Enter the form of message to display when the skill is used. The setting method is the same as [Normal] type skills.

22. Switch Toggle ON/OFF

·Specify the switch to turn ON after the skill is used. Click the ... button to open the [Switches] window, and specify the switch to turn ON from the list.
