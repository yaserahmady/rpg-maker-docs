Create Project

[Project] → [New Project]

 

·To create a new game, follow these steps to create a project*:

 

1.Select [New Project] from the [Project] menu.

 

2.The [New Project] window will be displayed. Here, there are 3 options to set before the project can be created.

 

Folder Name

The name of the folder for project management. The limit is 40 characters.

Game Title

Shown in the editor as well as during gameplay in the window title bar. The limit is 40 characters.

Project Base Path

Specifies the project folder location. To set the location, click on the [...] button and choose from the displayed folder tree.

 

3.When the [OK] button is clicked, the project will be created based on the above settings. In the map data list, a folder with the game title will be displayed, along with a new map.

* It is not possible to create a project with the same base path and folder name as an existing project.
