Name Input Processing

[Name Input Processing]

 

Function

·Opens the Name Input Processing.

·The player will be able to name actors in-game.

 

Settings

·Under [Actor], the actor whose name is to be changed can be chosen from the dropdown menu.

·[Initial Entry Mode] is used to specify whether the initial naming screen is in [Letters] or [Symbols].

·If [Show Default Name] is selected in [Option], the original name will already be entered in the name field when the name entry screen is shown.

 

Notes

·Controls for name entry are as follows:

Direction Keys

Move cursor

Select Key

Add the character that the cursor is on to the name

Cancel Key

Remove the last character of the name
