Running and Quitting the Game

·To end the game, choose To Title Screen from the main menu of the game, then choose Shutdown from the title screen. When playing in windowed mode, the game can also be closed by clicking the [X] button at the top-right corner of the title bar. Another way to end the game is by pressing the [Alt] and [F4] keys simultaneously.

* Number keys refer to the keys on the number pad.

* When using a joystick or gamepad, the directional pad serves as the directional keys, buttons 1 and 3 are the select keys, and button 2 is the cancel key.

 

·The functions of each key range from manipulating the character on the map, to choosing options from message windows. These are the main functions:

oCancel key

Opens the menu screen.

Cancels one action, or closes the menu screen (*1).

 

·Depending on the game events, the [Shift] key can also be used.

 

·The function keys have the following uses.

[SHIFT]: This will enable the player to skip dialogue when in playtest.

[CTRL]: In playtest, this will enable the player to walk through obstacles such as walls and unpassable tiles and prevent enemy encounters.

[F4]: Toggles fullscreen/windowed modes.

[F5]: Changes window size. the small window (320x240) is mostly useful for 1:1 screenshots.

[F9]: Opens the Debug Menu when in playtest.

[F10]: Cancels the currently running event (autostart events are restarted) when in playtest.

[F12]: Returns to the title screen. During battle test mode, it restarts the battle.

 

In-Game Commands (*2)

 

Items: Use items.

Skills: Use skills.

Equipment: Equip items.

Save: Save game progress.

To Title: Return to title screen.

 

·Commands that can be selected on the battle screen are as follows:

 

A.Party command. Select an action for the whole party to take.

 

Fight: Battle manually.

Auto: Battle automatically (skips character command selection and attacks repeatedly).

Escape: Attempt escape from battle.

 

B.Character command. If Fight is selected as a party command, the actions of each character can be specified.

 

Attack: Attack with currently equipped weapon.

Skills: Use created skills.

Item: Use an item.

Defend: Defend against attacks, reducing damage taken.

 

*1

In situations when an option has already been processed, such as after using an item or after entering a battle, the action cannot be canceled.

 

*2

Command names may differ from game to game (changeable from the [Terms] tab in the [Database] window). The default names will be used to explain here.
