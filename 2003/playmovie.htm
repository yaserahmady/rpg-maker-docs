[Play Movie]

 

Function

Plays a movie (AVI file, MPEG file).

 

Settings

 

 

·The movie to play is set under [Movie file]. Click the [...] button, choose the file name from the [Movie file] window, and click the [OK] button.
 

·[Display size] specifies the size that the movie will be played at. The left field specifies the horizontal width (1~320 pixels) while the right field specifies the vertical height (1~240 pixels).

oUnder [Display Position], the coordinates for the top-left corner of the movie can be set using either of the methods below.

oConstant: Displays the movie at a fixed position defined by its top-left coordinates. Enter the horizontal [X] coordinate (0~320) and vertical [Y] coordinate (0~240).

oVariable: Uses values stored in variables to specify the top-left corner of the movie. The variables used can be specified from the [Variables] window accessible by clicking the [...] buttons.
