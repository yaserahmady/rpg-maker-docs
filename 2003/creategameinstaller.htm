[Main Window] → [Supplementary Features] → [Create Game Installer]

 

·This feature creates a game installer that contains all the data required to play a completed game. With the game installer, even users who do not have this software can play games made with it.

·The creation process is as follows:

 

1.In [Game Installer Path] in the [Create Game Installer] window, specify the location to save the created game installer. Click on the [...] button to open the [Game Installer Creation Path] window, and use the mouse to select the appropriate folder.

 

 

2.If you want to include the Runtime Package (RTP) with the game data, enable the [Include Runtime Package in game disk] option.

 

3.Click the [Create] button to create the game installer based on the above settings. After processing is complete, the files that make up the game installer will be saved in the specified folder.

 

·When distributing your game, all files within the folder created by this feature should be packaged together.

 

Using Game Installers

·Before playing a game in game installer form, setup is required.

·Setup is carried out as follows:

 

1.Run [setup.exe] included in the game installer (the icon in the picture may differ based on the computer system).
