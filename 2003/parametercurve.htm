[Database] → [Actors] → [Parameter Curve]

 

·Actors' stats at each level are specified using the Parameter Curves.

·From Maximum HP, Maximum MP, Attack, Defense, Mind and Agility, open the desired tab.

·The displayed graph is drawn based on the current settings. The horizontal axis represents the level count, while the vertical axis represents the value of the stat at the respective level. Stats can be set either manually or automatically.

·3 methods can be used to edit the settings. Here, [Maximum HP] will be used to explain the setting methods. Other stats can also be set in similar manners.

 

Manual Setting

Stats at each level will be directly specified (the maximum HP stat will be used as an example here).

1.Enter the level to set for under [Level] within a range of 1 to 99. The current setting for the entered level will be displayed under [Maximum HP].

 

2.Under [Maximum HP], enter the new value within the range of 1 to 9999.

3.In the same vein, set the stats for each level.

 

oThe stat values can also be changed by clicking and dragging the bar graphs for each level.

 

Automatic setting

 

1.Generate Curve

a.Automatically generates the stats for each level based on the values entered for levels 1 and 99.

b.Click the [Generate Curve] button. The [Generate Curve] window will open (see below).

c.Enter the minimum and maximum stat values under [Level 1] and [Level 99] respectively, within a range of 1 to 9999.

d.Using the [Growth Type] slider, specify the growth type. The closer the slider is to Fast, the smaller the stat growth as level increases. On the other hand, the closer the slider is to Slow, the larger the stat growth as level increases.

e.Click the [OK] button to close the window. Based on the contents of the settings, the stats at each level will be set automatically.

 

2.Simple Settings

·Values provided by this software will be used. Clicking any of the Genius Type, Superior Type, Standard Type, Inferior Type buttons will set the respective stat growth type with random stats within a certain range.
