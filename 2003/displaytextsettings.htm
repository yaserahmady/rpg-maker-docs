[Display Text Settings]

 

Function

·Changes the display format for text.

 

Settings

 

·Under [Window display format], choose one of the following 2 options for the message window display format.

Normal:

Shows the frame, and text in the center.

Transparent

Does not show the frame, displaying only the text.

 

·Under [Window display position], choose one of the following 3 options for the window's display position.

Top

Top of screen

Center

Center of screen

Bottom

Bottom of screen

 

·Under [Options], choose display options and/or performance during wait.

Prevent the window from obscuring the player

If enabled, the window will be moved to the bottom of the screen if the player is at the top of the screen and vice versa.

Allow other events to move

By default, during wait status such as when messages are being displayed, events that move will not be processed.
If this option is enabled, events will continue to move even during wait time.

 

Notes

·Settings changed by this command will remain in effect even after the event processing is complete.

·To return the display format to its original state after event processing is complete, this command needs to be used again in the last line of the event to change the settings again.
