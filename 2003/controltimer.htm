[Control Timer]

 

Function

·Operates a timer.

·This command makes it possible to execute an event after a certain amount of time has passed.

·Up to 2 timers can be used at the same time.

 

Settings

·Under [Target Timer], use radio buttons to choose from either of [Timer 1] or [Timer 2].

 

·Under [Operation], choose the timer operation to execute with this command.

Set Time

Specify the time on the timer.

Start Timer

Starts the timer (Start).

Stop Timer

Stops the timer (Stop).

 

·[Operand] only needs to be set when [Set value] is chosen under [Operation]. Use either of the following commands to set the time on the timer.

Constant

Specify a fixed time. Enter the time in units of [minutes] [seconds].

Variable

The number stored in the given variable will be used as the time in seconds. For example, if variable no. 2 is specified which contains 5 at the time the command is processed, the timer will be set to 5 seconds.

The variable to be used can be specified by clicking the [...] button to bring up the [Variables] window.

 

·[Options] can only be enabled when [Start timer] is chosen under [Operation]. Specify whether to use any of the following two functions with regards to the timer.

Show Timer

Displays the timer on the screen.

Continue in Battles

The timer continues to count even during battle.

 

Notes

·The timer stops counting when the menu is displayed.
