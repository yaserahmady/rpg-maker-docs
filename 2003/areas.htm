About Areas

·Areas are imaginary spaces on the map that can have special settings and functions. These settings are called area data.

·Area data allows for different enemy groups to be set separately from the map data. Enemy groups set using map data will be encountered all over the map. However, enemy groups set using the area data will appear only within those specific boundaries. In other words, it is possible to set a habitat (encounter area) on a map.

·Area data can be used to define locations for [Transfer]-type skills. It is possible to provide several destinations on a single map.

Creating Area Data

·Area creation is done as follows:

 

1.From the map data list, right click the desired map and choose New Area from the shortcut menu.

 

2.The Area Creation window will open. Area boundaries and encounterable enemy groups can be specified here. For specific information, please refer to Area Creation.

 

3.Once the required settings have been made, click the [OK] button to close the window. The area data settings will be displayed on the map data list.

 

·Upon clicking the area data option from the map data list, area boundaries will be displayed in the map display.
