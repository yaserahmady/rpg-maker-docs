[Wait]

 

Function

·Sets a period of time when the player will not be able to control the character.
For events such as moving the character or making changes to the screen where time is needed, the event will be processed while the player has no control.

 

 

Settings

 

·There are 2 options for the wait time that can be chosen via radio buttons.

oFor [Time], enter the amount of time to wait. The time is measured in units of 0.1 seconds.

oFor [Wait for Key Input], wait is applied until the player presses any key.

 

Notes

·The effect of the wait command is the same as the effect when a message is displayed onscreen using [Show Text].
