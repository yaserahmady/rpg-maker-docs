About this Software

·This software is a game construction tool for role-playing games (RPGs), a popular genre of videogames.

·Many options for game content are provided in the editor. Just by entering values as required, you can create a game even without programming knowledge.

·Basic game resources (such as graphics, sounds, etc.) are provided in the Runtime Package (RTP), so you can start making a game immediately. Of course, custom resources can be imported as well.

·Games made with RPG Maker 2003 can be played by users who do not have this software as long as they download the RTP.
