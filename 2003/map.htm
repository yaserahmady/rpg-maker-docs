[Main Window] → [Menu Bar] → [Map] menu

 

Save Map Data

·Saves the map data for the current project.

 

Revert to Last Save

·Discards new edits and reloads the last save.

·A message will appear to confirm if the edits should be discarded. If this is desired, click [Yes]. Clicking [No] will return you to the original screen without doing anything.

 

Edit Mode

·Changes the map edit mode. [Lower Layer], [Upper Layer], and [Events] can be selected from the submenu.

 

Scale

·Changes the map zoom level. The zoom display level can be chosen from [1/1 Times], [1/2 Times], [1/4 Times], and [1/8 Times].
