[Move Picture]

 

Function

·Changes the position, zoom, transparency and/or effects of a displayed picture. A transition period can be specified in order to move the picture over the screen or change its effect properties gradually.

 

Settings

 

 

1.Number, Display Position, Picture Settings, Zoom, Transparency

·Under [Number], select the ID of the picture whose properties you want to change.

·Please refer to [Show Picture] for a detailed descriptions of the options under [Number], [Display Position], [Picture Settings], [Zoom] and [Transparency].

oIt is not possible to modify the [Scroll Picture with Map], [Picture Graphic], [Spritesheet] and [Use Transparent Color] options from a [Move Picture] command. Use [Show Picture] instead.

 

2.Time

·Under [Time], enter the duration of time used to change the picture display gradually (e.g. move the picture across the screen) in units of 0.1 seconds. The range is 0~1000. Use zero to change the picture's properties to their new values instantly.

 

3.Operation

·If [Wait for Completion] is enabled under [Operation], event processing will not continue until processing of this command is complete.
